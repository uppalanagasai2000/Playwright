# Playwright Tests

This repository contains Playwright test examples.

Quick start:

1. Install Git (if not installed).
2. Install dependencies:

```powershell
cd C:\WorkspacePlaywright
npm install
npx playwright install
```

3. Run tests:

```powershell
npx playwright test
```

To push to GitHub:

```powershell
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/uppalanagasai2000/Playwright.git
git branch -M main
git push -u origin main
```
